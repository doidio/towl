# G-THA

一句话主题：生成式 AI 端到端预测准确填充匹配患者术前 CT 的 THA 股骨柄假体

## 背景

概述 THA 手术，描述股骨柄假体与股骨髓腔填充匹配的重要性及其术前规划的意义

#### 一解到多解的思维容量升级

1. 股骨柄植入过程具有随机性，医生最后一击打开了薛定谔的猫的盖子
2. 手术规划的目标不是寻求唯一解，在客观上存在多解可能性，是有关患者个体的条件概率事件
3. 人不能两次进行同一个手术，但在计算机模拟的平行宇宙中却可以生成多种术后结果

#### 有招到无招的决策范式升级

1. 智能驾驶领域正在从基于规则的决策算法走向多模态端到端的大脑升级
2. 医学影像领域也迎来了判别式 AI 到生成式 AI 的方法论升级
3. 过往骨科 AI 术前规划可归纳为基于测量的决策，测量本身仍属于规则驱动，也必将走向无具体规则的纯数据驱动

## 相关研究

1. 生成式扩散模型在骨外科学的应用研究很少，有一些用于其它外科假体生成的研究但方法差异较大，例如颅骨缺损修复假体的自动生成
2. 最相关的是 THA-Net 论文，主要实现了 Inpainting 生成，但致命缺陷是选框内的骨骼和假体均为自由创作，生成结果临床不可用
3. 所以本项目最主要的工作是通过严格控制骨与假体界面条件，实现了临床准确的股骨柄假体生成

## 核心技术路线

1. VAE 编码术前 CT 作为生成条件，采用通道拼接的方式注入
2. VAE 编码术后 CT 提取的假体作为生成目标
3. LDM 潜在空间扩散模型学习在术前 CT 条件下生成术后 CT 中的假体

#### 生成条件中的骨质结构增强

一些其它研究通常在术前 CT 中使用截断阈值提取骨骼表面，丢失了原始图像信息，是不推荐的做法。

为了生成的骨与假体界面准确，模型需要学习尽可能丰富的骨质特征，原始图像 CT 值能够充分反映假体表面附近丰富的骨骼密度变化， 
因此原始体素场更有利于学习骨骼结构。

采用分段线性映射归一化，主要集中在`[150, 650]`范围的骨松质到骨皮质增强映射到`[-1, 0]`，
同时`[650, 3071]`范围的高密度骨皮质压缩映射到`[0, 1]`，直接舍弃`[-1024, 150]`范围的软组织、空气等假体无关内容，
以尽可能增强条件通道编码与假体相关的图像特征

#### 生成目标中的假体界面隐式条件

一些其它研究通常采用二值化掩膜提示生成，例如 Inpainting 指定一个多边形区域在内部生成，或使用 Mask 加权训练阶段的 Loss，
实质上端到端生成模型退化成了规则强相关的系统，是不推荐的做法。

直接生成术后 CT 也是不推荐的做法，骨骼到金属的密度跃升、金属伪影的针状亮暗带，都是 LDM 难以学习的特征。

为了使模型更关注骨与假体界面，也就是术后 CT 中高密度金属假体表面附近对应的术前 CT 骨骼，到底哪些骨质在 THA 术中被磨锉，哪些骨质被压实，
哪些骨质紧贴假体表面，哪些骨质与假体中间留有空隙，采用 SDF 距离场表示假体几何形状，并且截断到`[-5, 5]`mm，归一化映射到`[-1, 1]`，
假体作为生成目标，本身就是一种最佳的隐式条件，引导模型更关注 TSDF 截断距离场的窄带内部。同时 TSDF 是光滑的，非常适合 LDM 学习，
也适合引入 Eikonal 程函损失，以约束 VAE 的数值准确性。

## 数据集准备

回顾性收集共 1483 个患者，经过人工筛选排除不符合要求的数据，最终数据集容纳了 1366 个术区，
同一患者如果双侧手术可以提取两个术区。

以能够刚性配准术前术后假体所在区域为原则，具体纳排标准：

1. 纳入：
   1. 髋关节炎、股骨颈骨折等常见初次全髋置换适应症
   2. 股骨头坏死内固定，术前股骨头钢钉不影响假体依赖的骨界面
   3. 骨折内固定，术后钢圈、钢钉不直接接触假体
   4. 术前 CT 扫描范围有限导致股骨不完整，希望能仅参照部分骨质生成完整假体
2. 排除：
   1. 术前骨折、术后复位导致骨与假体界面破坏变形
   2. 术后股骨小转子下截骨，通常为了调节腿长
   3. 术后钢板、髓内钉等较长金属，干扰股骨柄区域
   4. 术后 CT 扫描范围有限导致假体不完整，不希望生成残缺的假体
   5. 半髋置换

#### 抗伪影的像素级配准

紧密接触骨质的假体部分要看术前骨质条件生成，其余假体部分要看自身条件生成，前者属于图像到图像的翻译任务，术前骨质属于空间条件。

骨骼形态很容易高精度、像素级配准是硬组织影像相比软组织影像的天然优势，所以通道拼接相比 ControlNet 是更简单有效的空间条件注入方式。
术前骨质作为条件通道与噪声拼接，模型在第一层卷积即可同时捕获条件和目标，学习更迅速，推理更轻量。

因此，术前术后 THA 术区的像素级配准是非常必要的。

核心方法是通过配准术前术后 CT 的共性骨面子集，实现骨面全集的刚性配准。骨面是 CT 体素之间插值提取的等值面，因此重建精度是亚像素级。
使用 GPU 加速的 Dual-MC 方法提取高质量的等值面，再使用 GPU 加速的 ICP 方法精确配准。

ICP 方法是点云、表面高精度配准的经典方法，通常效果不佳的原因在于使用不当，包括：

1. 初始配准太差，导致局部伪收敛
2. 采样点云存在特异性，导致结果向干扰因素妥协

对于 THA 术区的骨面刚性配准，术前术后 CT 体位基本一致，因此可以保证较好的初始配准。必须抵抗的干扰因素包括：

1. 术前特异性：
   1. 要手术截除的股骨头
   2. 要手术移除的内固定钢钉等
   3. CT 扫描相比术后更长的部分
2. 术后特异性：
   1. 金属假体
   2. 金属伪影
   3. CT 扫描相比术前更长的部分

最主要的干扰来源于术后金属假体，因此参与配准的点云采样必须避开金属伪影。术后 CT 很容易通过高阈值提取金属内芯，再建立距离场，
仅允许远离金属内芯 5mm 以上的区域采样，即可有效避开金属伪影。

此外，由于参与配准的点云采样不同，ICP 的迭代指标 MSE 并不能直接评价配准的优劣，必须人工确认术前术后骨面之间的重合情况，
尤其是股骨小转子等未经手术改变且远离金属伪影的重要解剖的对齐情况。

为了 GPU 加速 ICP 迭代过程，使用 `warp.Mesh` 重构了最频繁调用的 `trimesh.proximity.closest_point` 空间查找算法，
配准时长从分钟级缩短到毫秒级，极大加速了大规模数据集准备进程。

## 实验

实验环境是 1 个 NVIDIA RTX A6000 48GB，核心依赖库：

```requirements
torch==2.9.1+cu126
monai==1.5.2
monai-generative==0.2.3
diso==0.1.4
itk==5.4.4
trimesh==4.10.0
warp-lang==1.11.0
```

金属假体 VAE 的训练结果：

```commandline
Epoch:	 40
Params:	 0.01 B
L1:  	 0.008195408843457699
PSNR:	 39.553550720214844
SSIM:	 0.9945306181907654
Scale:	 5.771659900564642
```

术前骨质 VAE 的训练结果：

```commandline
Epoch:	 48
Params:	 0.01 B
L1:  	 0.015178157333284616
PSNR:	 31.9530086517334
SSIM:	 0.9624795317649841
Scale:	 2.91504249358613
```

在双 VAE 都获得极佳质量的情况下，LDM 训练了 1000 轮，观测其学习过程能明显发现：

1. 约 100 轮，股骨柄学会了填充匹配股骨远端髓腔，同时紧贴股骨近端内侧高亮的压应力骨小梁
2. 约 200 轮，股骨柄学会了股骨近端外侧大粗隆区域形成规则的 CAD 几何外形
3. 约 300 轮，股骨柄球头和髋臼杯明显更圆
4. 约 600 轮，股骨柄球头和髋臼杯内部更致密
5. 约 900 轮，股骨柄微调了内外翻姿态，学会了匹配骨质与保持自身形状之间的权衡

由于采用了 EMA 平均权重，同时 LDM 的训练十分稳定，模型快速收敛之后经历了漫长的后期微调，CFG 是调控生成质量与条件约束之间博弈的关键。
由于通道拼接的强力约束，模型率先学会了紧贴骨质边界，不可避免的生成了弯曲变形的股骨柄，随后在 CFG 引导下不断提升自身形状质量。

此外，虽然股骨柄区域是像素级配准的，但股骨颈截骨面以上的股骨头无法严格配准，髋臼侧骨盆也存在不确定的关节运动变化，这会造成困惑吗？

有趣的是，卷积特征提取本身就是一个分类器，结合注意力选择性衰减的能力，模型很容易学会识别哪里应该填充匹配，哪里应该生成正常的 CAD 几何外形。

```commandline
Epoch:	 999
Params:	 0.04 B
State:	 EMA
CFG:	 3.0 Guided
DDIM:	 50
```
